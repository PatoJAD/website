Se realiza un fork de PairDrop para crear VShare, una plataforma para compartir archivos de manera rapida y sencilla en la red local. El sitio esta construido con Bun. En este sitio realizamos:

* Se pule interfaz de usuario
* Se reduce el tamaño de la imagen del sitio
* Se agrega historial de archivos compartidos
* Se agrega la posibilidad de eliminar archivos del historial
* Se agrega la posibilidad de eliminar archivos compartidos
